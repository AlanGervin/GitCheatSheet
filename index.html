Git cheat sheet





Command Description
git add -A -u If files have moved in the your project executing this from the root directory of your project will update the files in the project
git reset --cached filename remove filename from the staged area but doesn't remove it from the working folder
git reset --soft HEAD~1 will put changes back in the stage area up the number after HEAD~. this allows you to rewrite your commit history.
git reset --hard HEAD~1 will remove the last commit from the local branch before it is submitted making it appear as though it never happened in the project
git reset filename will remove the file from that staged area
